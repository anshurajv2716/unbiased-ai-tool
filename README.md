# ⚖️ Unbiased AI Decision Tool

### Google Solution Challenge 2026 | Team Anjani & Anshu

## 🎯 Problem Statement
Every day, millions of people are rejected by AI systems —
not because they lack skills, but because the AI is biased.
Hiring tools, loan systems, and healthcare AI all carry
hidden discrimination from historical data.

## 💡 Our Solution
A no-code web tool that allows ANY organization to:
- Upload their dataset (CSV)
- Detect bias automatically
- See Before vs After comparison
- Get actionable recommendations
- Download bias report

## 🔧 Tech Stack
- Python 3.13
- AIF360 (IBM Fairness Library)
- Streamlit (Web Interface)
- Pandas + NumPy (Data Processing)
- Matplotlib (Visualizations)
- scikit-learn (ML Models)

## 🚀 How to Run
```bash
pip install streamlit pandas numpy scikit-learn aif360 matplotlib
streamlit run 03_upgraded2p.py
```

## 📊 Features
- ✅ 3 Sectors — Job Hiring, Loan Approval, Healthcare
- ✅ Real CSV Upload
- ✅ Traffic Light Bias Indicator
- ✅ Before vs After Comparison
- ✅ Hindi Explanations
- ✅ PDF Report Download
